Homebrew-Science

Scientific formulae for the Homebrew package manager

View on GitHub Download .zip Download .tar.gz

These formulæ provide software tailored to scientific endeavours. Need to store large amounts of data in a sparse matrix, solve a huge optimization problem or process the images from your space telescope? Maybe we can help!

How do I install these formulæ?

brew install homebrew/science/<formula>

Or brew tap homebrew/science and then brew install <formula>.

Troubleshooting

IMPORTANT First read the Troubleshooting Checklist.

Use brew gist-logs <formula> to create a Gist and post the link in your issue.

Search open and closed issues. See also Homebrew's Common Issues and FAQ.
